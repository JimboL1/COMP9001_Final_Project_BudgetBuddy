## Title
**BudgetBuddy: Smart Student Budgeting Tool**

## Description (Copy & Paste for Padlet)
BudgetBuddy is a lightweight, Python-based monthly budgeting tool designed specifically for students. It features flexible budget allocation using the popular 50/30/20 rule or custom percentages, intelligent expense tracking with category management, and smart overspending detection. The tool provides personalized "shrink suggestions" based on spending patterns and offers visual analytics through matplotlib charts. All data is stored locally in CSV/JSON format ensuring complete privacy. Target audience includes university students, recent graduates, and young adults learning financial management. The intuitive CLI interface makes budgeting accessible while the smart suggestion system helps users develop better spending habits.
