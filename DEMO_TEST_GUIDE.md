# BudgetBuddy 演示测试指南 🎯

这份指南将帮助你展示BudgetBuddy的所有功能。按照步骤操作，可以完整演示所有10个新功能。

---

## 📋 快速开始

### 准备工作
1. 运行程序：
   ```bash
   python3 budget_buddy.py
   ```

2. （可选）如果已有测试数据，先清理：
   ```bash
   rm -rf data/
   ```

## 🎬 完整演示流程

### 第一部分：基础设置和支出管理

#### 步骤1：设置预算
**菜单选项：** 1
- 选择：`1` (50/30/20规则)
- 输入月收入：`2000`
- **预期结果：** 显示预算分配卡片（$1000 Needs, $600 Wants, $400 Savings）

#### 步骤2：添加几笔支出
**菜单选项：** 2

**添加以下支出：**
1. 类别：`rent`，金额：`800`，描述：`Monthly apartment rent`
2. 类别：`groceries`，金额：`150`，描述：`Weekly grocery shopping`
3. 类别：`utilities`，金额：`120`，描述：`Electricity and water bill`
4. 类别：`entertainment`，金额：`80`，描述：`Movie tickets`
5. 类别：`dining`，金额：`120`，描述：`Restaurant meals`
6. 类别：`transport`，金额：`60`，描述：`Bus and train passes`

**预期结果：** 每次添加后显示绿色的成功消息和支出详情卡片

#### 步骤3：查看月度汇总
**菜单选项：** 4
- 直接按Enter（使用当前月份）
- **预期结果：** 
  - 显示总支出和支出次数
  - 表格显示各类别支出和百分比

#### 步骤4：编辑支出
**菜单选项：** 3
- 查看支出列表表格
- 选择一个支出（例如第3个）
- 选择：`e` (编辑)
- 修改金额为：`100`
- **预期结果：** 支出被更新

#### 步骤5：删除支出
**菜单选项：** 3

## 🔧 故障排查

如果遇到问题：

1. **数据不足** - 某些功能（如预测）需要足够的历史数据
2. **图表不显示** - 确保matplotlib正常工作
3. **颜色不显示** - 某些终端可能不支持ANSI颜色，但功能正常
4. **文件权限** - 确保有data目录的写入权限
